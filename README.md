# ZipTax Reference

A reference repository for the [ZipTax Sales Tax API](https://developers.zip.tax) containing machine-readable files, documentation, configurations, code samples, and other materials for API integration.

## Purpose

This repository serves as a centralized location for:
- API specification files (OpenAPI/Swagger)
- Configuration templates and examples
- Sample code and integration examples
- Test data and validation resources
- Postman collections and environment files
- JSON schemas and data models

## Contents

Materials in this repository support developers integrating with the ZipTax API by providing reference implementations, test resources, and reusable configurations.

## API Documentation

For complete API documentation, visit [developers.zip.tax](https://developers.zip.tax)
